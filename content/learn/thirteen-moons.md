# 13:28周期序 {align="center"}

<p style="text-align:center;color:#888;margin-bottom:1.5rem">一年13个月，每月28天——回到太阳与月亮的自然节律</p>

**13:28周期序**是13月亮历的"日历"部分——它定义了一年如何划分、月份如何命名、每周如何组织。与需要每年调整的公历不同，13:28周期序是**完全规律的**：每个月都是28天（恰好4周），整年结构固定不变。

---

## 一、10月到3月的命名 {color="blue"}

### 每个周期的名称

每月28天，对应**4周**，每周一个固定的主题。

| 星期 | 名称 | 能量主题 |
|------|------|----------|
| 第1天 | 磁性 | 启动、设定意图 |
| 第2天 | 月亮 | 面对挑战 |
| 第3天 | 电力 | 服务、激活 |
| 第4天 | 自我存在 | 定义、结构 |
| 第5天 | 超频 | 力量、放射 |
| 第6天 | 韵律 | 平衡、组织 |
| 第7天 | 共振 | 调和、通道 |
| 第8天 | 银河 | 和谐、完整 |
| 第9天 | 太阳 | 意图、实现 |
| 第10天 | 行星 | 完美、显化 |
| 第11天 | 光谱 | 释放、放下 |
| 第12天 | 水晶 | 合作、普及 |
| 第13天 | 宇宙 | 超越、存在 |
| 第14天 | 磁性 | （重复第一周） |
| ... | ... | ... |

> 每7天（一周）对应一个"银河周"，共4个银河周构成一个月。

**无时间日（Day Out of Time）**
每年**7月25日**，不属于任何月份或星期，是纯粹的"时间之外"的一天。传统上用于：
- 艺术创作、音乐、舞蹈
- 与自然连接、静心
- 释放过去一年，迎接新年
- 宽恕、和解、庆祝生命

这一天也被称为"绿色日子"——宇宙能量的通道格外畅通。

## 三、每周7天 {color="blue"}

13月亮历中的一周也是7天，但每一天不再用"星期一、星期二"这样基于行星的命名，而是用基于调性的命名，同时每天也有对应的**等离子体**：

| 星期 | 名称 | 对应脉轮 | 意义 |
|------|------|----------|------|
| 第1天 | DALI（动力日） | 海底轮 | 启动、开端 |
| 第2天 | SELI（目标日） | 脐轮 | 设定意图 |
| 第3天 | GAMMA（行动日） | 太阳神经丛 | 采取行动 |
| 第4天 | KALI（平衡日） | 心轮 | 平衡、调整 |
| 第5天 | ALPHA（显化日） | 喉轮 | 表达、显化 |
| 第6天 | LIMI（净化日） | 眉心轮 | 清理、接收 |
| 第7天 | SILIO（整合日） | 顶轮 | 回归、休息 |

## 四、魔法乌龟（Magic Turtle）{color="blue"}

### 乌龟壳的秘密

在13月亮历中，**乌龟**是一个重要的象征。乌龟壳上正好有 **13片甲片**——正好对应13个月。

<div class="highlight">
**乌龟的寓意：**
- 龟壳上的13片甲片 → 一年13个月
- 28个边缘刻度 → 每月28天
- 乌龟行动缓慢但坚定 → 自然的时间节奏
- 龟甲上的几何纹理 → 宇宙的数学结构
</div>

### 如何画魔法乌龟

1. 画一个乌龟壳轮廓
2. 在壳上画出13片甲片（中央1片，周围12片）
3. 每片甲片代表一个月
4. 标注每个月的起始日（公历日期）
5. 在龟壳周围画28个刻度（代表每天的日期）

## 五、13:28周期序 vs 公历 {color="blue"}

| 维度 | 13:28周期序 | 公历 |
|------|------------|------|
| 每月天数 | 固定28天 | 28/29/30/31天（不等） |
| 每月周数 | 固定4周 | 4～5周 |
| 月数 | 13个月 | 12个月 |
| 新年 | 固定7月26日 | 1月1日 |
| 闰年 | 不需要（结构固定） | 每4年1次闰年 |
| 时间感知 | 循环、可预期 | 线性、不可预期 |

## 六、实际应用 {color="blue"}

### 日常使用建议

1. **月初设定意图** — 每个月第一天（磁性之月的相似调性），写下这个月的目标
2. **每周检视** — 每周第7天（SILIO），回顾一周的共时性事件
3. **月份转换觉察** — 每月最后1天，做一个小小的结束仪式
4. **无时间日特别安排** — 7月25日，专门留给艺术、静心、感恩

<div class="tip">
**注意**：13月亮历与公历完全兼容。你可以同时使用两者——公历用于社会约定（会议、deadline），13月亮历用于能量觉察和个人成长。
</div>
