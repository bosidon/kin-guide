13月亮历，全称 **13月亮/28天历**（13 Moon / 28 Day Calendar），是由美国艺术史学家、精神导师 **何塞·阿圭勒斯（José Argüelles，1939–2011）** 于1989年发展出的共时历法系统。

<div class="highlight">
**核心理念：时间即艺术（Time is Art）**

13月亮历挑战现代"时间就是金钱"的线性机械观，提出"时间是艺术"的宇宙观——时间不是被切割的刻度，而是与宇宙能量同步的创造性流动。
</div>

## 一、起源与精神 {color="blue"}

### 何塞·阿圭勒斯（José Argüelles）

- 1939年生于美国，芝加哥大学**艺术史与美学博士**
- 曾执教于普林斯顿大学、旧金山艺术学院
- 1970年代开始研究古玛雅历法与时间科学
- 1987年发起**谐波汇聚**（Harmonic Convergence）— 全球集体冥想活动
- 1989年与妻子洛伊丁（Lloydine）共同发展出**DreamSpell**时间法则系统
- 1992年出版《时间法则的转变》
- 2011年逝世于墨西哥特奥蒂瓦坎

### 历法的来源

13月亮历融合了三大元素：

1. **古玛雅卓尔金历（Tzolk'in）** — 260天周期（13 × 20）
2. **月亮周期** — 13个月，每月28天（对应女性的生理周期）
3. **阿圭勒斯的研究** — 结合帕卡尔·沃坦（Pacal Votan）石碑的天文信息

<div class="tip">
**注意**：13月亮历（DreamSpell）≠ 传统玛雅历法。传统玛雅历法使用哈布历（Haab'，365天）和长计历（Long Count）；DreamSpell是阿圭勒斯基于古玛雅智慧再创作的当代灵性历法系统，两者使用不同的计算公式。
</div>

## 二、核心公式：13:20 vs 12:60 {color="blue"}

13月亮历最基本的区分在于两个频率：

| 频率 | 代表 | 特征 |
|------|------|------|
| **13:20** | 自然时间 | 13个月×28天、260天卓尔金历 |
| **12:60** | 机械时间 | 12个月（不等长）、60分钟/60秒 |

**13:20 自然频率的特征：**

- 13 × 20 = 260（卓尔金循环）
- 基于银河周期、自然节律
- 鼓励活在当下、直觉、创造力
- 时间 = 艺术（体验的质量）

**12:60 人造频率的特征：**

- 12个月（28/29/30/31天不等） + 60进制时间
- 基于罗马儒略历的改造
- 鼓励线性、效率、生产主义
- 时间 = 金钱（交换的媒介）

<div class="highlight">
13月亮历旨在帮助人们从12:60的机械频率中**调频**到13:20的自然频率，从而更轻松地生活、更高效地创造。
</div>

## 三、基本结构 {color="blue"}

### 年度结构

```
一年 = 13个月 × 28天 = 364天 + 1天无时间日 = 365天
```

| 项目 | 数值 |
|------|------|
| 全年总天数 | 365天 |
| 每个月 | 28天（恰好4周） |
| 每个月数 | 13个月 |
| 无时间日 | 7月25日（不属于任何月/周） |
| 新年 | 7月26日 |

## 四、什么是 Kin？ {color="blue"}

**Kin（星系印记）** 是13月亮历中最基本的能量单位。

每个Kin由 **1个调性（Tone）+ 1个图腾（Seal）** 组成。

- 调性 = **怎么做**（动能/节奏/动作）
- 图腾 = **做什么**（能量/本质/品质）

```
Kin 1 = 磁性调性 + 红龙图腾 = 磁性红龙
Kin 2 = 月亮调性 + 白风图腾 = 月亮白风
     ...
Kin 260 = 宇宙调性 + 黄太阳图腾 = 宇宙黄太阳
```

<div class="highlight">
**你的生日Kin**（出生印记）是你此生的核心天赋频率。它不代表你的全部，但它是最容易让你与宇宙共时性对齐的能量入口。
</div>

## 五、与公历的关系 {color="blue"}

13月亮历与公历之间可以相互转换：

| 公历日期 | 13月亮历日期 |
|----------|------------|
| 7月26日 | 第1个月（磁性之月）第1天 |
| 8月22日 | 第1个月第28天 |
| 7月25日 | **无时间日**（不属于任何月份） |

> **计算公式**：Kin = ((365 × (公历年−1900) + 52 + 当年第几天 − 1) % 260) + 1
